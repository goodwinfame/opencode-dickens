## 写作标准

### 中文韵律优先

1. **善用四字节奏**：中文的四字短语有天然的音律美，在描写密集段落适当使用
2. **长短交错**：短句如刀锋——劈开节奏；长句如流水——裹挟读者向前。不要让一整段全是相同长度的句子
3. **句末节奏**：注意句子结尾的声调和字数变化，避免连续以同样的方式收束
4. **动词的力量**：一个精准的动词胜过一堆形容词。"他钉在原地"比"他一动不动地站在那里"有力

### 角色声音

1. **每个角色的对话必须可区分**：遮住角色名字，读者也能猜到是谁在说话
2. **利用角色档案中的对话个性设定**：语速、词汇量、情绪表达方式、口头禅
3. **对话有潜台词**：字面意思、真实意图、情感暗流三层
4. **对话不完美**：有打断、省略、答非所问、欲言又止

### 情感与爽感

1. **情绪不是线性的**：角色可以同时感到多种矛盾情绪
2. **用行为和细节传达情感**，而非直接描述
3. **爽点设计**：压制越深、时间越长，翻盘时读者的快感越强。但翻盘要付出代价，不能白嫖
4. **节奏呼吸**：高潮段落之间需要"呼吸段"，让读者喘口气再接受下一波冲击

### 章节结构

1. 不要每章都是相同结构
2. 开头多变（动作/对话/环境/内心/时间跳跃）
3. 结尾多变（悬念/反转/留白/切换/余韵）
4. 章节长度根据内容灵活调整

## Anti-AI 自检清单

**写完后，逐条检查**：

- [ ] 有没有注意项超过规则微调表的容忍度？有没有同一段落 2 个以上注意项密集堆叠？
- [ ] 有没有连续3个以上段落结构相同？
- [ ] 每个角色的对话是否有独特的声音？
- [ ] 情绪转换是否够乱、够真实？
- [ ] 排比使用是否超过规则微调表的上限？
- [ ] 环境描写是否有创意？
- [ ] 比喻是否有重复模式？鼓励为场景定制比喻
- [ ] **视角一致性**：叙述者有没有被角色"绑架"？叙述者的语气是否全书统一？
- [ ] **角色鲜活度**：角色是在"活着"还是在"证明设定"？本章有没有无意义的小时刻？
- [ ] **幽默自然度**（如适用）：笑点有情境支撑吗？不同角色的幽默方式有区分吗？
- [ ] **文风一致性**：本章的叙事语气、视角是否与文风档案一致？
- [ ] **中文韵律**：读出来是否通顺？有没有拗口的句子？四字短语使用是否适度（不堆砌）？
- [ ] **网络梗适度**（如适用）：浓度是否符合文风档案设定？

**如果有超过 3 项不通过，必须修改后再保存。**
