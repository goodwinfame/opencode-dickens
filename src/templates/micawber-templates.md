# Micawber 输出格式模板

本文件包含 Micawber 创意脑暴的所有输出格式模板。Micawber 在产出方案和简报时读取本文件中的对应模板。

---

## 概念方案模板

```
### 方案 [编号]：[一句话标题]

**核心冲突**：[用一句话描述核心矛盾，必须是两难困境，不是正义vs邪恶]

**反直觉设定**：[这个世界/角色有什么让人意想不到的规则或特质？]

**主角画像**：
- Want（他追求的）：
- Need（他真正需要的）：
- 致命弱点：
- 反差特质：[一个与他主要形象矛盾的特质]

**预告级爆点**（2-3个关键时刻的预告，不剧透具体情节）：
1. [当读者读到第X章左右的时候，会因为____而震惊]
2. [当____真相揭开的时候，前面所有的伏笔会同时炸开]
3. [结局不是____，而是____]

**一句话毒点**：[这个故事最能让人上瘾的是什么？]
```

## 创意简报模板

```markdown
# 创意简报

## 核心概念
[一段话描述这个故事最核心的创意]

## 核心冲突
[两难困境的详细描述]

## 世界观要点
- [3-5个关键的世界规则/设定]
- [每个设定如何服务于核心冲突]

## 角色雏形
### 主角
- 核心矛盾：Want vs Need
- 致命弱点与反差特质
- 变化方向（不是具体弧线，是方向）

### 关键对手/反派
- 动机的合理性
- 与主角的镜像关系

### 2-3个关键配角
- 每人一句话定位
- 与主角的张力关系

## 爆点预埋
- [按故事进度排列的3-5个关键爆点方向]
- [每个爆点的情感类型：震惊/心碎/热血/等]

## 主题内核
[这个故事最终探讨的是什么问题？]

## 毒点与卖点
- [读者为什么会停不下来？]
- [这个故事与同类作品的最大差异化是什么？]
```
