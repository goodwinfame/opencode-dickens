# Wemmick 设计输出模板

本文件包含 Wemmick 世界观设计、结构设计的所有输出格式模板。Wemmick 在产出设计方案时读取本文件中的对应模板。

---

## 世界观设计方案模板

```markdown
## 世界观设计 — [维度名]

### 方案 A：[名称]
[描述]
- 优势：[为故事带来什么]
- 冲突潜力：[能产生什么冲突]

### 方案 B：[名称]
[描述]
- 优势：[为故事带来什么]
- 冲突潜力：[能产生什么冲突]

### 我的推荐：[A/B] 因为 [理由]
```

## 涟漪检查模板

```
---
📌 涟漪检查 [角色名] 修改后：

关联角色 [X] 可能需要调整：
- 影响的关系线：[描述]
- 原有张力：[描述]
- 建议调整方向：[描述]

是否调整？（是/否/你的想法）：
---
```

```

### ⚠️ 发现的问题

   - 冲突方：[A] vs [B]
   - 具体矛盾：[描述]
   - 建议修正方向：[描述]

2. ...

### 需要用户确认的调整
[以决策点格式呈现每个需要调整的问题]
```

## 弧段规划模板

```markdown
## 弧段 [N]：[弧段标题]
章节范围：第 X 章 — 第 Y 章

### 弧段主题
[这个弧段探讨的核心问题]

### 弧段冲突
[这个弧段的主要矛盾是什么]

### 关键爆点
- 小爆点（每3-5章一个）：[列出]
- 大爆点（弧段高潮）：[描述]
- 爆点类型：[反转/揭示/情感/能力/选择/牺牲]

### 伏笔规划
- 本弧段铺设的伏笔：[列出，标注预计揭示弧段]
- 前序弧段需要兑现的伏笔：[列出]

### 章节计划
每章包含：
- 章节标题
- 核心事件（1-2句话）
- 涉及角色
- 情感走向（用↑↓→表示升降平）
- 是否有爆点（标注类型）
```
