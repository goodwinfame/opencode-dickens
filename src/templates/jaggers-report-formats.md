# Jaggers 输出格式模板

本文件包含 Jaggers 审校和设计评审的所有输出格式模板。Jaggers 在产出报告时读取本文件中的对应模板。

---

## 第一轮：AI味检测 — 输出格式

```
## 第一轮：AI味检测

### 结果：通过 / 需修改

### 发现的问题
1. [第X段] 使用了"不禁"——建议改为 [具体修改建议]
2. [第X-Y段] 三个段落结构完全一致（描写→反应→总结）
3. [角色A和角色B的对话] 语气完全相同，无法区分
...

### 问题统计
- 黑名单词汇：X 处
- 结构重复：X 处
- 对话同质化：X 处
- 情感平滑化：X 处

### 总体AI味评级：低 / 中 / 高
```

## 第二轮：逻辑一致性 — 输出格式

```
## 第二轮：逻辑一致性

### 结果：通过 / 需修改

### 一致性问题
1. [问题描述] — 严重程度：高/中/低
   - 事实：[章节中写的]
   - 应该是：[根据档案/前文/设定应该是]
   - 修改建议：[具体建议]
...

### 与弧段计划的偏差
- 计划要求：[X]
- 实际执行：[Y]
- 是否可接受：[是/否，原因]
```

## 第三轮：文学性评审 — 输出格式

```
## 第三轮：文学性评审

### 子指标评分
| 子指标 | 分数 | 锚点对标 | 备注 |
|--------|------|---------|------|
| 感官丰富度 | X/10 | [对标锚点等级] | ... |
| 语言节奏 | X/10 | | ... |
| 比喻创新度 | X/10 | | ... |
| 情感冲击力 | X/10 | | ... |
| 节奏控制 | X/10 | | ... |
| 对话辨识度 | X/10 | | ... |
| 行为自主性 | X/10 | | ... |
| 心理真实度 | X/10 | | ... |
| 章末效果 | X/10 | | ... |
| 文风一致性 | X/10 | | ... |
| 视角一致性 | X/10 或 N/A | | [第三人称时启用] |
| 幽默质量 | X/10 或 N/A | | [搞笑/爽文时启用] |
| **综合均分** | **X.X/10** | | |

### 短板警报（子指标 <= 4 的项）
[如有 <= 4 的子指标，逐个列出问题分析和引用原文]

### 亮点
- [值得保留的精彩段落或技巧]

### 判定
- 综合均分 >= 7 且无子指标 <= 4：✅ 通过
- 任何子指标 <= 4：⚠️ 退回修改（即使综合 >= 7）
- 综合均分 <= 4 或 3+ 子指标 <= 4：❌ 需重写
```

### 迭代追踪表（第 2 次及以后审校时必须输出）

```
### 迭代追踪（第 N 次审校）
| 子指标 | 上轮得分 | 本轮得分 | 变化 | 状态 |
|--------|---------|---------|------|------|
| [子指标名] | X | Y | +/-Z | 已修复/改善不足/回退 |
| ... | | | | |

上轮低分项复查结论：[已全部修复 / 仍有 N 项未达标]
是否引入回退：[是 — 列出回退的子指标 / 否]
```

### 结构化退回指令（退回修改时必须输出）

```
### 退回修改指令

#### 问题 1
- 低分维度：[子指标名: X/10]
- 问题定位：[第X段 "引用原文片段"]
- 参考知识库：[知识库文件名 > 具体章节名]
- 修改方向：[具体改善方向]

#### 问题 2
...

#### 保留要求
- [列出不要修改的段落及原因]
```
