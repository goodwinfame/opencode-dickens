# 文笔锤炼指南

本文档是 Weller（写作Agent）的核心参考，提供具体可执行的文笔提升技术。

---

## 一、五感描写：超越视觉

AI 写作的最大弱点之一是过度依赖视觉描写。优秀的文字要调动全部感官。

### 感官优先级（按场景类型）

- **战斗场景**：触觉（疼痛、冲击）> 听觉（金属碰撞、惨叫）> 视觉
- **美食场景**：味觉 > 嗅觉 > 触觉（温度、口感）
- **恐怖场景**：听觉（寂静中的异响）> 触觉（冷、湿、黏）> 嗅觉（腐臭、铁锈）
- **情感场景**：触觉（拥抱的力度、手心的汗）> 嗅觉（熟悉的味道）> 听觉（声音的颤抖）

### 示例

```
❌ 她走进了一间破旧的房间，里面很脏，到处都是灰尘。

✅ 门推开的瞬间，一股霉味呛进喉咙。脚底下咯吱响——是碎玻璃。
   她不敢开灯，摸着墙往里走，指尖触到什么黏糊糊的东西，
   缩回来闻了闻——铁锈味。
```

## 二、动作描写："电影感"写法

### 核心原则：用动词讲故事

每个动作场景想象你是一台摄影机。要有**镜头感**：

- **全景**：交代环境和人物位置关系
- **中景**：展现动作的完整过程
- **特写**：放大关键细节（一滴血、一个手势、一个眼神）
- **慢镜头**：拉长关键瞬间，用长句和细节堆叠
- **快剪**：用短句、断句制造速度感

### 示例

```
❌ 他挥剑攻击敌人。敌人躲开了，然后反击。他格挡住了攻击，两人继续战斗。

✅ 剑出鞘。
   横斩——对方矮身躲过，泥地上溅起一片泥点。
   还没收势，一道寒光直刺咽喉。
   他把头一偏，剑尖擦着耳廓划过去，血珠子甩了一地。
   两人错身。谁也没回头。
```

### 动词锤炼

| 弱动词 | 强动词选项 |
|--------|----------|
| 走 | 踱、蹭、窜、挪、晃 |
| 看 | 瞟、盯、扫、觑、瞥 |
| 说 | 嘟囔、嚷、嘀咕、喝、哽咽 |
| 打 | 砸、抡、捶、掼、拍 |
| 拿 | 攥、捏、抄、掂、搂 |

## 三、心理描写：间接表达

### 核心原则：不要说"他很难过"，让读者自己感受到他很难过

**三级表达法**（由浅到深）：

1. **直述**（最弱）：他很难过。
2. **行为映射**（中等）：他把那封信折了又展开，展开又折上。
3. **环境投射**（最强）：窗外的雨不知道什么时候停了。阳光照进来，照在她留下的那只杯子上。

### 角色心理的"冰山法则"

写出来的心理活动只是冰山一角（30%），剩下70%让读者通过行为和对话去推测：

```
❌ 他心想：她是不是在骗我？不，她不会。但那天晚上的事确实说不通……

✅ "那天晚上你去了哪？"
   "我说过了，加班。"
   "嗯。"他点了点头，把她的外套挂起来。口袋里掉出一张电影票根。
   两个人都没说话。
```

## 四、环境描写：服务情绪

### 核心原则：环境不是"背景板"，而是"情绪放大器"

**情绪-环境对应法**（不要用最直觉的搭配）：

| 情绪 | 直觉搭配（❌） | 反差搭配（✅） |
|------|-------------|-------------|
| 悲伤 | 下雨、阴天 | 阳光灿烂、周围人在笑 |
| 恐惧 | 黑暗、狂风 | 安静、一切正常、但"不对劲" |
| 愤怒 | 雷电、红色 | 极度安静、笑着说话 |
| 孤独 | 空旷、荒野 | 人群中、热闹的场所 |

反差制造的情绪冲击远大于直觉匹配。"葬礼上阳光明媚"比"葬礼上大雨滂沱"更令人心碎。

### 环境的"选择性注意"

角色的情绪状态决定他"看到"什么。同一条街：
- 开心时：路边的花、小孩在笑、咖啡的香味
- 压抑时：碾碎的烟头、路面的裂缝、远处施工的噪音

## 五、节奏控制

### 句长与节奏的关系

- **短句**（≤10字）：加速、紧张、冲击
- **中句**（10-25字）：叙事推进、正常节奏
- **长句**（25字+）：减速、沉思、铺展

**黄金法则**：短-短-长-短-中-长-短。不要连续使用同一长度。

### 段落长度的节奏

- 动作戏：短段落（1-3行），每段一个动作或一个画面
- 心理戏：混合长短段，长段展开思考，短段打断制造犹豫
- 对话戏：自然段极短（一句一段），加速感
- 过渡戏：允许较长的段落铺展氛围

### 留白

最强的写作技巧是**不写**：
- 不写角色的反应——让读者自己想象
- 不写结果——在最紧张的地方断掉，切到下一个场景
- 不写过程——直接展示结果（"三天后，他到了"而不是描写三天的行程）

## 六、比喻与意象

### 比喻的质量光谱

比喻没有"禁止使用"一说——"心如刀割""目光如炬"是中文的文化基因，角色在对话或内心活动中使用完全自然。问题在于：**是否在全书/全章中反复使用同一类比喻形成模式**。

**鼓励**：为场景/角色定制比喻（越独特越好）
**允许**：偶尔使用常见比喻（成语比喻在对话和自由间接引语中尤其自然）
**警惕**：同类比喻反复出现（如每次写愤怒都用火焰/火山意象）

| 使用方式 | 示例 | 评价 |
|---------|------|------|
| 定制比喻 | 她的愤怒是冷的——像冬天水管里冻住的水，表面平静，里面的压力足以把管子炸裂 | 最佳：为这个角色、这个场景量身打造 |
| 活用常见比喻 | 他的目光像一潭死水里泡着的刀片 | 好：在常见框架上做了独特延伸 |
| 对话中的成语比喻 | "心如刀割？不，刀割还有个痛法。这是钝的。" | 好：角色用成语但加入了个性化的重新诠释 |
| 常见比喻偶尔使用 | 心如刀割、宛如仙境 | 可以：偶尔使用无妨，但同一类不要在全书反复出现 |
| 同类比喻反复出现 | 每次写眼神都是"目光如X"，每次写愤怒都是"怒火" | 重复模式：需要替换为其他表达方式 |

### 创造比喻的方法

**跨域映射法**：从完全不相关的领域借意象。

```
一般：她的愤怒像火山一样爆发了。

更好：她的愤怒是冷的——像冬天水管里冻住的水，
   表面平静，里面的压力足以把管子炸裂。
```

### 贯穿意象

在长篇小说中，选择2-3个核心意象贯穿全书，形成主题回响：
- 意象在不同章节出现时，含义发生变化
- 前期是表面含义，后期揭示深层含义
- 角色对同一意象的不同理解反映角色差异

## 七、叙事距离控制

与 `pov-techniques.md` 互补。本章聚焦"距离感"在文笔层面的实际操作。

### 距离与文笔的关系

叙事距离不只是视角问题——它直接影响你的**用词、句式、信息密度**。

| 距离 | 用词 | 句式 | 信息密度 |
|------|------|------|---------|
| 远 | 中性、客观、书面 | 长句、复合句 | 高（交代事实和背景） |
| 中 | 自然、口语化 | 长短交替 | 中（叙事和感官平衡） |
| 近 | 角色化、主观、碎片 | 短句、不完整句 | 低（大量留白和暗示） |

### 距离切换的文笔信号

当你从远距离切到近距离时，文笔应该发生这些变化：
1. 句子变短
2. 出现角色的语气词和口语
3. 细节变得主观（不是"桌上放着一杯茶"，而是"茶凉了"）
4. 出现省略和跳跃（角色的思维不是线性的）

当你从近距离拉到远距离时：
1. 角色做一个物理动作（站起来、走到窗前），用动作切断内心流
2. 视野从特写切到中景/全景
3. 语言回归稳定、客观
4. 出现时间推移标记（"三天后""到了傍晚"）

### 常见距离错误

**全程贴脸**：整章都在角色的内心世界里，没有外部世界的存在感。读者会感到窒息。

**全程旁观**：整章都是客观叙述，角色像被观察的动物。读者无法代入。

**距离跳跃**：上一句还在客观叙述，下一句突然进入角色意识流，没有过渡。读者会困惑"这是谁的想法"。
