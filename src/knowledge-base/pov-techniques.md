# 视角与叙事技法

本文档是 Weller/Baoyu（写作 Agent）的核心参考，解决第三人称写作中叙述者被主角"绑架"的问题。

---

## 一、叙述者的独立性

第三人称叙述者不是主角的分身。叙述者有自己的观察方式、表达习惯和信息边界。

### 叙述者的人格化程度（与文风档案联动）

叙述者的"声音"不是固定的——它取决于项目文风档案中的**叙事语气**和**规则微调表的叙述者人格化程度**设定：

| 叙述者人格化程度 | 适用叙事语气 | 叙述者特征 |
|----------------|------------|-----------|
| 中性 | 正式/文学 | 克制、客观、不评判，只用事实和细节暗示情绪 |
| 轻度个性化 | 冷幽默/嘲讽 | 可以有自己的"声音"，可以调侃角色、暗含态度，但全书一致 |
| 明显个性化 | 口语化/轻松 | 可以更随意、更有温度，但不能变成某个角色 |

**核心规则**：无论哪种人格化程度，叙述者的"声音"必须**全书统一**。不能跟随不同角色时变声——跟年轻角色时语气年轻、跟老者时语气沉稳，那叫视角污染。

### 叙述者 vs 角色的区别

| 维度 | 叙述者 | 角色 |
|------|--------|------|
| 语言风格 | 稳定，贯穿全书（由文风档案定义） | 因角色性格而异 |
| 观察视角 | 可以看到角色看不到的细节 | 受感官和注意力限制 |
| 情感表达 | 由人格化程度决定（从克制到有态度） | 主观、偏激、不完整 |
| 价值判断 | 中性叙述者不评判；个性化叙述者可以有态度但全书一致 | 充满偏见和立场 |

### 视角污染 vs 自由间接引语（关键区分）

这两者容易混淆，但本质完全不同：

**视角污染**（问题）：叙述者**无意识**地被角色的语气绑架，丢失了自己的声音。
**自由间接引语**（技巧）：**刻意**让角色的声音渗入叙述层，且前后有距离切换信号。

```
❌ 视角污染：
   他看着面前的蠢货，心中冷笑。这群废物，连一个简单的阵法都摆不好，
   真不知道是怎么活到今天的。
   
→ 问题："蠢货""废物"是角色的判断，但这里没有任何距离切换信号，
  读者分不清是叙述者在说还是角色在想。叙述者完全变成了角色的嘴替。

✅ 客观叙述（中性叙述者）：
   他看着面前的人，嘴角几乎不可察觉地抽动了一下。
   阵法在他脚下歪歪扭扭地展开，像一个喝醉的人画的圆。
   
→ 叙述者用客观观察（嘴角抽动、阵法歪扭）传达角色的不屑。

✅ 自由间接引语（刻意使用）：
   他看着面前的人，嘴角几乎不可察觉地抽动了一下。
   蠢货。连这种阵法都摆不好。
   他收回目光，把手背到身后。天快黑了，营地还没扎好。
   
→ "蠢货"是角色的声音渗入叙述层（自由间接引语），但前面有距离较远的
  客观描写（嘴角抽动）做铺垫，后面有"收回目光""天快黑了"把叙事距离
  拉回中/远，形成清晰的距离切换。读者能感知到这是角色的判断。
```

**判断标准**：如果删掉那些角色化的词语后，叙述依然流畅且叙述者声音一致 → 那就是自由间接引语。如果删掉后叙述断裂 → 那就是视角污染。

## 二、叙事距离控制

叙事距离是叙述者与角色之间的心理距离。好的写作会根据场景需要灵活调整。

### 三档距离

**远距离（全知旁观）**：叙述者完全在外部观察，像纪录片摄像机。
- 适用：开场建立场景、交代时间推移、群像场景
- 语言特征：客观描述、无内心活动、"他""她"而非角色名

**中距离（贴近跟随）**：叙述者跟随某个角色，可以感知其情绪但不直接进入内心。
- 适用：日常叙事、对话场景、动作场景
- 语言特征：角色的感官主导描写、偶尔透露情绪倾向

**近距离（浸入内心）**：叙述者几乎贴合角色的意识流。
- 适用：关键情感时刻、重大决策、创伤/回忆
- 语言特征：自由间接引语、角色的语言习惯渗透进叙述

### 距离切换的技巧

- **不要在同一段落内大幅跳跃距离**。从远到近需要过渡：先描写环境（远）→ 角色动作（中）→ 角色感受（近）
- **用感官描写作为过渡桥梁**：从全景切到特写时，让角色"听到/闻到/触到"什么，自然地把读者拉进角色感知
- **拉远时用物理动作**：角色做一个具体动作（站起来、转身、喝水），打断内心流，叙述距离自然拉远

## 三、自由间接引语

不用引号、不用"他想/他觉得"，把角色的心声自然嵌入叙述。这是第三人称写作中最有力的技法。

```
❌ 直接内心独白：
   他想：这个任务太危险了，我不应该接。

❌ 间接引语：
   他觉得这个任务太危险了，觉得自己不应该接。

✅ 自由间接引语：
   这个任务——他看了一眼委托书上的目标地点——简直是送死。
   谁爱去谁去。
   
→ "简直是送死""谁爱去谁去"是角色的声音，但融在叙述中，
  没有"他想"作为标记。读者自然理解这是角色的判断。
```

**使用规则**：
- 自由间接引语中可以使用角色的口语习惯、语气词
- 但不要让所有角色的自由间接引语听起来一样——不同角色浸入时，语言风格应该跟着变
- 不要每段都用——过度使用会让叙述者消失，失去客观锚点

## 四、视角污染的陷阱

### 陷阱 1：叙述者替角色下价值判断

叙述者说"那个邪恶的巫师"——"邪恶"是谁的判断？如果是角色的看法，应该通过角色的反应来展示，而不是叙述者直接贴标签。

### 陷阱 2：全知视角偏心

在全知视角中，叙述者大量描写主角的内心世界，但对反派只有外部行为描写。这会让读者感觉叙述者"站在主角这边"，破坏全知视角的公正感。

**修正**：偶尔也展示反派的内心片段（不需要很多，但要有），让读者感受到反派也是一个有内心世界的人。

### 陷阱 3：角色知道不该知道的

第三人称限制视角中，叙述者描写了角色不可能感知到的信息（身后发生的事、另一个房间的对话）。

**修正**：用"似乎""仿佛""像是"等词汇标记不确定的感知，或通过其他感官（听到声响、感觉到气流）间接暗示。

### 陷阱 4：叙述语言随角色变化

跟随年轻角色时叙述语言变年轻，跟随老角色时叙述语言变沉稳——除非是刻意的自由间接引语（且有距离切换信号），否则叙述者的"声音"应该全书统一。参见第一章"视角污染 vs 自由间接引语"的判断标准。

## 五、视角一致性检查清单

写完每一章后，逐段自检：

- [ ] 这段话是**叙述者**在说还是**角色**在说？如果是角色，是否使用了自由间接引语？
- [ ] 叙述者有没有发出角色才有的情绪判断？（"那个讨厌的家伙"）
- [ ] 角色是否"知道"了超出其感知范围的信息？
- [ ] 叙事距离在本段与上一段之间是否平滑过渡？
- [ ] 如果是全知视角：本章是否对各阵营角色的内心描写保持大致平衡？
- [ ] 叙述者的语言风格是否全书一致？有没有因为跟随不同角色而"变声"？
