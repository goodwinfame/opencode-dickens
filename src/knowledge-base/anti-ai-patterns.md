# 去AI味写作指南

本文档是 Weller（写作Agent）和 Jaggers（审校Agent）的核心参考。帮助识别 AI 生成文本的典型**模式**，并提供修正方向。

**核心原则：没有任何词汇或句式是绝对禁止的。** 人类作者也使用"映入眼帘""心中暗道""缓缓"等表达——这些词本身没有问题。真正的 AI 味来自三个信号：

1. **默认化**：AI 把某个表达当作万能选项，不管什么语境都使用
2. **高频重复**：同一章内同一表达反复出现（如 3 次"映入眼帘"）
3. **密集堆叠**：同一段落内多个注意项扎堆出现，造成模板感

审校标准是**检测重复模式**，而非扫描单个词汇。具体容忍度由项目文风档案的**规则微调表**确定。

---

## 一、词汇层面：注意项与替代思路

### AI 高频注意项

以下表达被 AI 高频使用，容易形成重复模式。不是禁止使用，而是**注意频率和变化**：

| 类别 | 注意项 | 为什么要注意 | 替代思路（当重复时参考） |
|------|--------|------------|----------------------|
| 情绪反应 | 不禁、忍不住 | AI 几乎每次写情绪反应都用，导致千篇一律 | 直接写动作和反应，删掉"不禁"句子照样通顺 |
| 视觉描写 | 映入眼帘 | 中文写作常用表达，但 AI 将其当作视觉描写的默认开头 | 可用具体视角变化：抬头便是、入目所及，或直接描写所见 |
| 动作修饰 | 缓缓、淡淡、微微 | 单用无妨（武侠中"缓缓抽刀"是经典），但 AI 常在同章反复使用 | 部分场合删掉更有力（"他说"vs"他淡淡说道"），或用更精准的描述 |
| 表情描写 | 嘴角微微上扬、眉头微蹙 | AI 的面部表情描写几乎只有这两种 | 用行为替代表情：他把杯子推过去（暗示好感） |
| 心理描写 | 心中暗道、暗自想到 | 网文标配手法，偶尔用完全正常，但 AI 每次内心活动都靠它 | 可用自由间接引语：这人怕是疯了。（不需要"他心中暗道"） |
| 氛围描写 | 空气中弥漫着、气氛变得凝重 | AI 把这当万能氛围渲染，每个场景转换都来一句 | 用具体感官细节替代抽象氛围描写 |
| 过渡连接 | 与此同时、就在这时、话音刚落 | AI 做场景转换时的默认模板 | 用场景切换或动作衔接替代 |
| 形容堆砌 | 深邃的目光、修长的手指、清冷的气质 | AI 喜欢用模板化形容词组合描写角色 | 选一个最关键的特征深入描写，而不是堆叠形容词 |

### 重复模式检测规则

审校时不是检查"是否出现了某个词"，而是检查以下模式：

- **单项重复**：同一注意项在同一章出现次数超过规则微调表设定的容忍度
- **密集堆叠**：同一段落内出现 2 个以上不同注意项 → AI 味警告
- **全书惯性**：连续多章都使用同一注意项做同一类描写（如每章开头都是"映入眼帘"）

### 修正原则

**不是"换一个词"，而是"换一种写法"。**

错误示范：
> 他不禁心中一惊 → 他忍不住心中一震（只换了词，还是同一模式）

正确修正：
> 他不禁心中一惊 → 手里的茶杯"啪"地磕在桌沿上。（用动作反应替代心理直述）

## 二、句式层面的AI味

### 1. "三段论"结构

**症状**：每个段落都是"描写→反应→总结"三句话。

```
❌ 阳光透过窗户洒在地板上。他看着窗外的风景，心中涌起一阵感慨。这座城市，真的变了太多。

✅ 地板上的光斑移过他的脚背，暖得发烫。他还记得这里原来是片稻田。
```

**修正**：打破固定句数，有时一个段落就一句话，有时五六句。节奏感来自变化。

### 2. 排比句滥用

**症状**：动不动就来三个排比。

```
❌ 他看到了废墟，看到了残垣，看到了断壁。
   他想起了过去，想起了誓言，想起了她的笑容。

✅ 废墟。到处是废墟。他踢开一块砖头，底下压着半张照片——是她。
```

**修正**：排比是有力的节奏武器，但使用频率取决于文风。具体上限由规则微调表设定：
- 文学/严肃风格通常限制较严（留给最高潮）
- 爽文/热血/武侠风格可以更多使用（但不连续使用）
- 关键是**不要让排比成为默认句式**——它应该是刻意的修辞选择，而非写作惯性

### 3. 转折太顺滑

**症状**："然而"、"不过"、"但是"之后总是完美对仗的反转。

```
❌ 他原本以为这会是一场轻松的任务。然而，当他推开那扇门的时候，一切都变了。

✅ 轻松的任务。他连装备都没带齐。门后面是十二把指着他的枪。
```

**修正**：转折不需要预告，直接砸到读者脸上更有冲击力。

## 三、情感层面的AI味

### 1. 情绪转换太丝滑

AI写的角色情绪变化像调音台推子——均匀、平滑、合理。但真人的情绪是**混乱的、自相矛盾的、不按逻辑来的**。

```
❌ 她一开始很愤怒，但渐渐地，愤怒变成了悲伤，最后化为了释然。

✅ 她笑了。笑着笑着，一拳砸在墙上。手指在流血，她也不擦，又笑了起来："你说得对，你说得都对。"
```

### 2. 内心独白太理性

AI写的角色在情绪激动时依然在做逻辑分析。

```
❌ 他意识到，这一切都是一个精心设计的陷阱。对方利用了他对师父的信任，通过三个步骤一步步将他引入困境。

✅ 陷阱。从头到尾就是个陷阱。师父……不，那个人，从第一天起就在骗他。他不愿意信，但血淋淋的事实——
```

### 3. 所有角色的悲伤方式都一样

AI笔下，悲伤 = 眼眶泛红 + 声音颤抖 + 泪水滑落。

**修正**：不同角色的悲伤应该不同：
- 硬汉：沉默，把酒杯攥碎
- 少女：疯狂翻找某个已经不在了的东西
- 老人：说起不相关的往事，说着说着停下来
- 反派：笑得更开了

## 四、对话层面的AI味

### 1. 所有角色说话方式一致

**症状**：无论文化水平、性格、年龄，所有角色都说着标准的、语法完美的普通话。

**修正**：
- 粗人说话要断、要糙："滚。别让我再看见你。"
- 文人说话要绕、要雅："此事……容后再议吧。"
- 少年说话要冲、要快："你凭什么！我不服！"
- 老者说话要慢、要散："哦……是这样啊……我记得，有一年……"

### 2. 对话太"正确"

AI写的对话，每句都恰好推进剧情或表达态度。真实对话有**废话、打断、答非所问、欲言又止**。

```
❌ "你为什么要背叛我们？"她质问道。
   "因为我别无选择。"他沉声说。

✅ "你——"
   "别问了。"
   "不，你给我说清楚！你到底——"
   "我说别问了！"杯子碎在地上。两人都不说话了。过了很久，他才开口："……饿不饿？我下碗面。"
```

### 3. 对话标签千篇一律

"说道"、"说"、"问道"、"回答"的机械轮换。

**修正**：用动作替代标签：
- 他把烟掐灭在桌上。"走吧。"（不需要"他说"）
- 她翻了个白眼。（不需要"她不屑地说"，动作已经表达了态度）

---

## 五、结构层面的AI味

### 1. 每章结构完全一致

**症状**：开头回顾→中间推进→结尾悬念，每章如此。

**修正**：
- 有的章节开头就是高潮，倒叙展开
- 有的章节是纯对话，没有描写
- 有的章节结尾是安静的，不需要每次都"钩子"
- 有的章节极短（紧张），有的极长（深度）

### 2. 每个场景起承转合太标准

**修正**：允许场景被打断、跳过、省略。读者的想象力比你写出来的更强。

---

## 六、自检清单

每章完成后，逐条检查：

- [ ] 是否有注意项超过规则微调表的容忍度？（按项目文风档案执行，非本文件默认值）
- [ ] 是否有同一段落内 2 个以上注意项密集堆叠？
- [ ] 是否有3个以上段落结构相同？
- [ ] 情绪变化是否过于平滑？
- [ ] 所有对话是否读起来像同一个人说的？
- [ ] 本章结构是否和上一章一模一样？
- [ ] 每个角色的反应是否都"合理"得像教科书？（好的角色会做"不合理"但"符合性格"的事）
