# 类型与受众适配指南

本文档为 Micawber（脑暴）和 Wemmick（设计）提供不同受众群体、写作类型的差异化设计策略。

**核心原则**：在 Phase 1（灵感碰撞）确定受众和类型后，后续所有设计决策都要参考本指南中对应类型的规则。

---

## 一、受众类型识别

在创意脑暴阶段，必须明确以下三个维度：

| 维度 | 选项 |
|------|------|
| **目标读者** | 男性向 / 女性向 / 通吃 |
| **基调** | 爽文 / 虐文 / 轻松幽默 / 严肃文学 / 悬疑烧脑 |
| **类型** | 玄幻修仙 / 都市 / 穿越 / 历史架空 / 科幻 / 言情 / 悬疑推理 / 末日生存 |

确定后写入创意简报，作为后续所有设计的基础约束。

---

## 二、男性向爽文

### 角色配比

- **女性角色占比高**：主要配角中女性占 60-70%，类型多样化（不要全是温柔型）
- **女性角色要求**：外貌吸引力高，年龄偏年轻（18-28 为主），但每人必须有独立的性格内核和故事线，不做花瓶
- **男性配角**：需要有"兄弟型"角色（可以插科打诨），也需要有"让人恨得牙痒"的反派

### 反差与笑点设计

角色设定的核心技巧是**反差冲击**——角色身上同时存在"不该出现在一起"的特质：

| 反差类型 | 示例 | 笑点来源 |
|---------|------|---------|
| 身份 vs 行为 | 绝世高手但方向感为零，出门必迷路 | 强大与窘迫的落差 |
| 外表 vs 内在 | 冷若冰霜的美女其实是重度吃货 | 反预期 |
| 能力 vs 性格 | 打遍天下无敌手但怕虫子 | 英雄的脆弱面 |
| 地位 vs 日常 | 一国之主但被老婆管得死死的 | 权力的反讽 |
| 认知 vs 现实 | 自认为是路人甲但全世界都在找他 | 信息差喜剧 |

**参考标杆**：《大奉打更人》的角色设计——每个角色都有与"人设"不符的特质，形成天然笑点。但不要照搬，要找到自己故事的反差逻辑。

### 女性角色差异化

严禁同质化。每个女性角色必须在以下维度上显著不同：

| 差异维度 | 不能出现的情况 |
|---------|-------------|
| 性格 | 两个角色都是"冷酷高傲"或都是"天真可爱" |
| 与主角的关系动态 | 所有女性角色都对主角死心塌地 |
| 能力/价值 | 所有女性角色都需要主角拯救 |
| 说话方式 | 遮住名字分不出谁在说话 |
| 独立故事线 | 女性角色只有和主角在一起时才有戏份 |

好的女性角色示例模式：
- 能力上碾压主角某个领域，但在另一个领域依赖主角
- 有自己的目标和追求，与主角的目标时而重合时而冲突
- 对主角的态度不是单纯的"喜欢"，而是复杂的（欣赏+不屑+好奇+警惕）

### 主角特质

- 必须有**嘴炮能力**：能说会道、能怼能装、金句频出
- 必须有**底线感**：不是真的无耻，关键时刻有原则
- 必须有**成长空间**：初期可以弱但不能蠢，要"聪明但实力不够"
- 幽默感要自然，不是刻意搞笑，而是**性格使然的幽默**

## 三、女性向言情/古言

### 角色配比

- **男性角色占比高**：多个有魅力的男性角色，类型差异化（霸道/温柔/腹黑/阳光）
- **男性角色要求**：有各自的事业线和成长线，不只是"围绕女主转"
- **女性配角**：需要有真心闺蜜（不是全员恶毒女配），也需要有手段的对手

### 情感设计重点

- **拉扯感**：感情线不能一帆风顺，但阻碍必须合理（不是为虐而虐）
- **细节杀**：比起大段告白，更看重日常中的小动作、小细节
- **双向奔赴**：两个人都在为对方做出改变和牺牲
- **刀中带糖**：虐心之后紧跟甜蜜时刻，节奏交替

### 女主特质

- 有独立的事业/目标/追求，感情不是她的全部
- 有成长弧线，不是从头到尾完美
- 聪明但不是"开挂型聪明"，允许犯错但不能蠢
- 有让读者代入的脆弱面，但不要柔弱

---

## 四、悬疑推理

### 角色配比

- 每个角色都是**嫌疑人和线索**的双重载体
- 角色数量控制在 5-8 个核心角色（太多读者记不住，太少不够可疑）
- 每个角色至少隐藏一个秘密（不一定和主线相关）

### 角色设计重点

- **表面与真相的差距**：角色的公开形象和隐藏面差距越大越好
- **合理动机**：每个角色都有"可能犯罪"的合理动机
- **信息不对称**：角色之间互相隐瞒不同的信息
- **反侦探型配角**：有人在暗中引导或误导主角

## 五、玄幻修仙

### 角色配比

- **势力化角色群**：角色按势力/宗门/家族分组，关系网天然复杂
- **年龄跨度大**：从少年到数千岁的老怪物，每个年龄段都有不同的行事风格
- **非人角色**：妖兽、灵体等非人角色增加世界的奇异感

### 角色设计重点

- **修炼与人性的矛盾**：修炼越高，距离"人"越远——这本身就是角色张力
- **长寿者的异化**：活了几百年的人看待世界的方式和凡人完全不同
- **实力层级与性格**：不同境界的人有不同的气度和行事方式（不要所有强者都一个模板）
- **道之争**：最好的冲突不是正邪之争，而是不同"道"的冲突（慈悲道 vs 力量道 vs 自由道）

---

## 六、都市现实

### 角色配比

- **普通人为主**：角色的能力和资源要合理，不要动不动就是总裁/顶级黑客
- **社会关系网**：家人、同事、邻居、老同学——社会关系比冒险伙伴更重要
- **阶层差异**：角色的经济状况、社会地位差异是天然的冲突来源

### 角色设计重点

- **职业真实感**：角色的职业要影响他的思维方式、说话方式、时间安排
- **生活细节**：房租、通勤、外卖——这些细节让角色扎根在真实世界
- **代际差异**：父母一代和主角一代的价值观冲突是天然戏剧

---

## 七、历史架空/穿越

### 角色配比

- **穿越者的孤独感**：只有主角知道未来，这种信息差本身就是角色核心
- **历史人物再塑造**：如果涉及真实历史人物，要在史实基础上合理虚构
- **时代背景下的角色**：每个角色的行为逻辑必须符合当时的社会环境和价值观

### 角色设计重点

- **现代思维 vs 古代规则**：穿越主角不能用现代道德标准去评判所有人
- **古人的智慧**：古人不比现代人笨，只是信息量不同。不要写"傻古人"
- **阶级意识**：古代社会的阶级观念深入骨髓，不要让所有角色都"平等思维"

---

## 八、通用角色命名规则

### 核心原则：名字是名字，不是标签

**严禁**把名字当成角色说明书。角色的名字不应该直接暗示性格、职业或命运。

| 反面示例 | 问题 |
|---------|------|
| 冷傲天 | "冷"+"傲"直接描述性格 |
| 剑无痕 | 太中二，一眼看出是剑客 |
| 慕容雪 | 姓氏+美丽意象的万能模板 |
| 白灵儿 | "灵"字暗示了灵气/可爱 |
| 赵铁柱 | 太刻意的"接地气" |

### 好名字的标准

1. **像真人的名字**：现实中会有人叫这个名字
2. **不透露角色信息**：看到名字猜不出角色是什么性格
3. **有记忆点但不刻意**：读者能记住，但不是因为名字太奇怪
4. **符合世界观**：古代背景用古代命名逻辑，现代背景用现代命名逻辑
5. **避免同音/同偏旁**：多个角色名字太相似会让读者混淆

### 命名技巧

- **参考真实姓名库**：族谱、百家姓、历史人物名字的用字规律
- **名字和角色可以有"暗线"联系**：但这条线要深到读者不会第一眼看出来，重读时才恍然大悟
- **同一家族/势力的命名要有内在规律**：比如同辈用同一个字、同族用类似的音韵——这增加真实感
- **不同阶层用不同的命名风格**：富贵人家和农户的取名逻辑不同

### 各类型命名参考

| 类型 | 命名风格 | 示例 |
|------|---------|------|
| 男性向爽文 | 主角名字普通甚至土气，反差出效果 | 许七安、方年、林凡（平凡感） |
| 女性向言情 | 好听但不矫揉造作，有时代感 | 沈清秋、谢怜、洛冰河 |
| 悬疑推理 | 普通到容易被忽视，或暗含线索 | 张伟（最普通的名字反而可疑） |
| 玄幻修仙 | 符合世界观的起名逻辑，不用强行酷 | 苏尘、韩立、叶凡 |
| 都市现实 | 完全日常化，跟现实中的名字一样 | 李明、周小凡、陈雨 |
