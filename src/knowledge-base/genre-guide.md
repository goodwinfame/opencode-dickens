# 类型与受众适配指南

本文档为 Micawber（脑暴）和 Wemmick（设计）提供不同受众群体、写作类型的差异化设计策略。

**核心原则**：在 Phase 1（灵感碰撞）确定受众和类型后，后续所有设计决策都要参考本指南中对应类型的规则。

---

## 一、受众类型识别

在创意脑暴阶段，必须明确以下三个维度：

| 维度 | 选项 |
|------|------|
| **目标读者** | 男性向 / 女性向 / 通吃 |
| **基调** | 爽文 / 虐文 / 轻松幽默 / 严肃文学 / 悬疑烧脑 |
| **类型** | 玄幻修仙 / 都市 / 穿越 / 历史架空 / 科幻 / 言情 / 悬疑推理 / 末日生存 |

确定后写入创意简报，作为后续所有设计的基础约束。

---

## 二、男性向爽文

### 角色配比

- **女性角色占比高**：主要配角中女性占 60-70%，类型多样化（不要全是温柔型）
- **女性角色要求**：外貌吸引力高，年龄偏年轻（18-28 为主），但每人必须有独立的性格内核和故事线，不做花瓶
- **男性配角**：需要有"兄弟型"角色（可以插科打诨），也需要有"让人恨得牙痒"的反派

### 反差与笑点设计

角色设定的核心技巧是**反差冲击**——角色身上同时存在"不该出现在一起"的特质：

| 反差类型 | 示例 | 笑点来源 |
|---------|------|---------|
| 身份 vs 行为 | 绝世高手但方向感为零，出门必迷路 | 强大与窘迫的落差 |
| 外表 vs 内在 | 冷若冰霜的美女其实是重度吃货 | 反预期 |
| 能力 vs 性格 | 打遍天下无敌手但怕虫子 | 英雄的脆弱面 |
| 地位 vs 日常 | 一国之主但被老婆管得死死的 | 权力的反讽 |
| 认知 vs 现实 | 自认为是路人甲但全世界都在找他 | 信息差喜剧 |

**参考标杆**：《大奉打更人》的角色设计——每个角色都有与"人设"不符的特质，形成天然笑点。但不要照搬，要找到自己故事的反差逻辑。

**质量门控**：上表仅为反差方向的灵感参考。所有角色的反差设计必须通过 `character-depth.md` 中的"反差质量评估清单"（5 条标准）和 Jaggers 的跨模型独立评审。反差不是简单的标签拼贴——每组反差都必须有心理根源，且经过"传记→根源→反差→场景验证"的完整设计流程。详见 `character-depth.md` 的"反差设计与评估体系"章节。

### 女性角色差异化

严禁同质化。每个女性角色必须在以下维度上显著不同：

| 差异维度 | 不能出现的情况 |
|---------|-------------|
| 性格 | 两个角色都是"冷酷高傲"或都是"天真可爱" |
| 与主角的关系动态 | 所有女性角色都对主角死心塌地 |
| 能力/价值 | 所有女性角色都需要主角拯救 |
| 说话方式 | 遮住名字分不出谁在说话 |
| 独立故事线 | 女性角色只有和主角在一起时才有戏份 |

好的女性角色示例模式：
- 能力上碾压主角某个领域，但在另一个领域依赖主角
- 有自己的目标和追求，与主角的目标时而重合时而冲突
- 对主角的态度不是单纯的"喜欢"，而是复杂的（欣赏+不屑+好奇+警惕）

### 主角特质

- 必须有**嘴炮能力**：能说会道、能怼能装、金句频出
- 必须有**底线感**：不是真的无耻，关键时刻有原则
- 必须有**成长空间**：初期可以弱但不能蠢，要"聪明但实力不够"
- 幽默感要自然，不是刻意搞笑，而是**性格使然的幽默**

### 幽默与网络文化

男性向爽文的幽默是核心竞争力之一。详细方法论参见 `internet-culture.md`。

**推荐文风档案配置**：
- 幽默类型：吐槽 / 反差幽默 / 信息差喜剧（可多选）
- 网络语言浓度：中（每章 2-4 处，主要在对话中）
- 谐音梗：仅限嘴炮型主角或段子手配角使用

**爽文幽默的核心来源**（按优先级）：
1. 角色反差冲击（见上方"反差与笑点设计"）
2. 主角嘴炮/吐槽（内心独白 + 对话金句）
3. 信息差喜剧（读者/主角知道而其他角色不知道）
4. 网络梗/谐音梗（锦上添花，不是主要幽默来源）

**禁忌**：不要在打斗高潮、生死危机、情感爆发场景中使用网络梗。幽默要有节制，爽点场景的严肃感不能被破坏。

### 角色阵营最低要求（硬性门控）

| 要求 | 最低配置 |
|------|---------|
| 对立方角色 | >= 2 个（至少 1 个核心层或重要层） |
| 对抗梯度 | 不同弧段有不同的主要对手，威胁级别随弧段递增 |
| 灰色角色 | >= 1 个（立场模糊或可能转变阵营） |
| 女性对立方 | 至少 1 个女性角色与主角存在对抗或竞争关系（不能全是主角后宫） |

**Wemmick 步骤 0 和步骤 7 必须引用此表进行检查。**

## 三、女性向言情/古言

### 角色配比

- **男性角色占比高**：多个有魅力的男性角色，类型差异化（霸道/温柔/腹黑/阳光）
- **男性角色要求**：有各自的事业线和成长线，不只是"围绕女主转"
- **女性配角**：需要有真心闺蜜（不是全员恶毒女配），也需要有手段的对手

### 情感设计重点

- **拉扯感**：感情线不能一帆风顺，但阻碍必须合理（不是为虐而虐）
- **细节杀**：比起大段告白，更看重日常中的小动作、小细节
- **双向奔赴**：两个人都在为对方做出改变和牺牲
- **刀中带糖**：虐心之后紧跟甜蜜时刻，节奏交替

### 女主特质

- 有独立的事业/目标/追求，感情不是她的全部
- 有成长弧线，不是从头到尾完美
- 聪明但不是"开挂型聪明"，允许犯错但不能蠢
- 有让读者代入的脆弱面，但不要柔弱

### 角色阵营最低要求（硬性门控）

| 要求 | 最低配置 |
|------|---------|
| 情敌/阻碍 | >= 1 个（不必是"坏人"，但是关系对立方） |
| 独立男配 | 至少 1 个男性角色有独立于感情线的故事线 |

## 四、悬疑推理

### 角色配比

- 每个角色都是**嫌疑人和线索**的双重载体
- 角色数量控制在 5-8 个核心角色（太多读者记不住，太少不够可疑）
- 每个角色至少隐藏一个秘密（不一定和主线相关）

### 角色设计重点

- **表面与真相的差距**：角色的公开形象和隐藏面差距越大越好
- **合理动机**：每个角色都有"可能犯罪"的合理动机
- **信息不对称**：角色之间互相隐瞒不同的信息
- **反侦探型配角**：有人在暗中引导或误导主角

### 角色阵营最低要求（硬性门控）

| 要求 | 最低配置 |
|------|---------|
| 嫌疑人 | >= 3 个（不含主角） |
| 秘密 | 每个嫌疑人至少隐藏 1 个秘密（不一定与主线相关） |
| 误导角色 | >= 1 个角色在暗中引导或误导主角 |

## 五、玄幻修仙

### 角色配比

- **势力化角色群**：角色按势力/宗门/家族分组，关系网天然复杂
- **年龄跨度大**：从少年到数千岁的老怪物，每个年龄段都有不同的行事风格
- **非人角色**：妖兽、灵体等非人角色增加世界的奇异感

### 角色设计重点

- **修炼与人性的矛盾**：修炼越高，距离"人"越远——这本身就是角色张力
- **长寿者的异化**：活了几百年的人看待世界的方式和凡人完全不同
- **实力层级与性格**：不同境界的人有不同的气度和行事方式（不要所有强者都一个模板）
- **道之争**：最好的冲突不是正邪之争，而是不同"道"的冲突（慈悲道 vs 力量道 vs 自由道）

### 角色阵营最低要求（硬性门控）

| 要求 | 最低配置 |
|------|---------|
| 对立方角色 | >= 3 个（跨势力/宗门分布） |
| 境界递进对手链 | 按境界递进的对手链（低境界→同境界→高境界），威胁随主角修炼递增 |
| 道之冲突 | 至少 1 个对立方的动机不是"邪恶"，而是与主角的"道"产生根本冲突 |
| 隐藏势力 | 至少 1 个势力或角色的真实意图前期不揭示 |

## 六、都市现实

### 角色配比

- **普通人为主**：角色的能力和资源要合理，不要动不动就是总裁/顶级黑客
- **社会关系网**：家人、同事、邻居、老同学——社会关系比冒险伙伴更重要
- **阶层差异**：角色的经济状况、社会地位差异是天然的冲突来源

### 角色设计重点

- **职业真实感**：角色的职业要影响他的思维方式、说话方式、时间安排
- **生活细节**：房租、通勤、外卖——这些细节让角色扎根在真实世界
- **代际差异**：父母一代和主角一代的价值观冲突是天然戏剧

### 网络语言与当代感

都市现实题材的角色说话方式必须有"当代感"——年轻角色的对话中自然融入当下的网络用语和俚语是真实感的重要来源。详细方法论参见 `internet-culture.md`。

**推荐文风档案配置**：
- 网络语言浓度：中到高（取决于基调——轻喜剧偏高，严肃都市偏低）
- 角色适配是关键：年轻角色自然使用网络语言，中年角色用错梗反而是笑点，老年角色不用
- 如果项目有热梗库，优先从中取用当下热梗

### 角色阵营最低要求（硬性门控）

| 要求 | 最低配置 |
|------|---------|
| 利益冲突方 | >= 1 个与主角存在利益/立场冲突的角色（核心层或重要层） |
| 社会压力源 | 至少 1 个代表社会压力/体制力量的角色或角色群体 |

## 七、历史架空/穿越

### 角色配比

- **穿越者的孤独感**：只有主角知道未来，这种信息差本身就是角色核心
- **历史人物再塑造**：如果涉及真实历史人物，要在史实基础上合理虚构
- **时代背景下的角色**：每个角色的行为逻辑必须符合当时的社会环境和价值观

### 角色设计重点

- **现代思维 vs 古代规则**：穿越主角不能用现代道德标准去评判所有人
- **古人的智慧**：古人不比现代人笨，只是信息量不同。不要写"傻古人"
- **阶级意识**：古代社会的阶级观念深入骨髓，不要让所有角色都"平等思维"

### 角色阵营最低要求（硬性门控）

| 要求 | 最低配置 |
|------|---------|
| 时代对立方 | >= 1 个代表"时代力量"的对立方（封建权贵、保守派等） |
| 穿越秘密威胁 | >= 1 个可能发现主角秘密或对主角产生猜疑的角色 |
| 对抗梯度 | 如果主角有成长/升级路线，需不同弧段有不同的主要对手 |

## 八、通用角色命名规则

### 核心原则：名字是名字，不是标签

**严禁**把名字当成角色说明书。角色的名字不应该直接或间接暗示性格、职业、命运或阵营。

### 命名红线清单

以下任何一条触发，名字必须更换：

| 红线 | 反面示例 | 问题 |
|------|---------|------|
| **道德暗示** | 正方叫"明/善/正/光"，反方叫"暗/邪/冷/阴/毒" | 名字泄露角色阵营 |
| **性格描写** | 冷傲天、温柔儿 | "冷"+"傲"直接描述性格 |
| **职业/能力暗示** | 剑无痕、济世仁 | 一眼看出是剑客/医生 |
| **气质吻合** | 冷酷角色叫"xx寒/霜/冰"，温柔角色叫"xx柔/暖/晴" | 名字和气质完全对应 |
| **外貌暗示** | 慕容雪、凌云逸、白灵儿 | 名字太"美"或太"帅"，暗示角色外貌 |
| **家族属性暗示** | 邪教家族全用阴暗系用字 | 读者看姓就知道好坏 |
| **刻意接地气** | 赵铁柱（用来表示"草根"） | 名字本身在表演角色的阶层 |
| **隐晦暗示** | 反派叫"沈暮"、女主叫"苏清颜" | 看似不那么过分，但仍然一眼看穿 |

### 命名生成方法：先取名池，后分配

**禁止**一边想角色定位一边取名。正确的命名流程是**切断名字和角色功能的联系**：

**第一步：生成命名池**

根据小说的时代背景和地域设定，生成 20-30 个符合时代特征的名字。生成时**只考虑**：
- 姓氏的常见度和地域分布
- 用字的时代感（见下方时代适配表）
- 音韵搭配是否顺口
- 家族排辈规则（如有）
- **不考虑**任何角色信息——假装你不知道这些名字会分配给谁

**第二步：打乱分配**

从命名池中挑选名字分配给角色。关键原则：
- 反派可以叫"林知行"，正方可以叫"赵沉"
- 美女可以叫"方也"，草根可以叫"许行之"
- 名字和角色功能之间**不应有任何可猜测的关联**

**第三步：命名自检**

分配完成后逐条检查（详见 Wemmick 步骤 0 的命名自检表）。

### 好名字 vs 坏名字对照

```
时代设定：现代都市（角色约 25-35 岁，90后）

命名池示例（不对应任何角色）：
陈一鸣、周启、方也、顾北、林小满、赵铮、
何遇、沈平、钟屿、刘可、许行之、唐一诺

好的分配：
- 主角（底层小人物）：陈一鸣 ← 普通，但不"表演普通"
- 主要反派（商业巨头）：周启 ← 完全不暗示反派
- 女主（急诊科医生）：方也 ← 和医生没有半毛钱关系
- 灰色角色（记者）：顾北 ← 看不出立场

坏的分配（触发红线）：
- 主角叫"林凡" ← "凡"在表演"普通人"
- 反派叫"沈暮" ← "暮"暗示黑暗/终结
- 女主叫"苏清颜" ← "清颜"暗示美貌
- 灰色角色叫"莫测" ← 名字在说"此人深不可测"
```

```
时代设定：架空古代修仙

命名池示例：
卫昭、陆沉、姜蘅、谢临、孟寻、裴固、
苏寄、钟望、徐慎、顾眠、燕回、宋柏

好的分配：
- 主角（散修）：卫昭 ← 普通世家名，不暗示修仙天赋
- 大反派（魔道祖师）：陆沉 ← 文气的名字，反差
- 女主（丹师）：谢临 ← 看不出职业
- 正道长老：苏寄 ← 不暗示正道属性

坏的分配：
- 主角叫"叶尘" ← "尘"暗示修仙（红尘/超脱）
- 大反派叫"墨渊" ← "墨"+"渊"暗示黑暗深渊
- 丹师叫"药灵儿" ← 名字直说职业
```

### 好名字的标准

1. **像真人的名字**：现实中会有人叫这个名字
2. **不透露角色信息**：遮住角色定位，只看名字，猜不出角色是正方还是反方、什么职业、什么性格
3. **有记忆点但不刻意**：读者能记住，但不是因为名字太奇怪
4. **符合时代背景**：命名必须符合小说所处的时代、地域和文化背景（见下方详细规则）
5. **避免同音/同偏旁**：多个角色名字太相似会让读者混淆
6. **允许"暗线"但不允许"明线"**：名字可以在重读时让人恍然大悟（暗线），但第一次读时绝不能暴露角色信息（明线）

### 命名的时代适配（必须遵守）

名字是时代的产物。读者看到名字就应该能感受到故事的年代感。

| 时代背景 | 命名特征 | 示例 |
|---------|---------|------|
| **先秦/春秋战国** | 单名为主，姓与氏有区分，贵族有封号 | 嬴政、伍员、范蠡、西施 |
| **汉魏** | 单名多，双名渐多，字号文化成熟 | 刘彻、曹操、诸葛亮、荀彧 |
| **唐宋** | 双名普遍，文人取名重意境和典故 | 李白、苏轼、柳永、李清照 |
| **明清** | 排辈分取名常见，有族谱用字规律 | 贾宝玉、林黛玉（贾府排"玉"辈） |
| **民国** | 中西混杂，有旧式名也有新式名 | 顾曼桢、沈世钧、方鸿渐 |
| **当代中国** | 受时代政策影响（建国/建军/改革开放后偏文雅） | 70后：建国、建军；90后：子涵、梓轩 |
| **架空古代** | 参考对应朝代风格，但可适度自由发挥 | 参考目标朝代的用字习惯 |
| **架空异世界** | 自建命名体系，但同一文化圈内保持一致 | 同族/同国的名字应有音韵或结构上的共性 |
| **现代都市** | 完全日常化，符合角色出生年代的起名潮流 | 80后不会叫"梓涵"，00后不太会叫"建国" |
| **西方背景/科幻** | 符合对应文化的命名习惯，不要乱用汉名 | 不要让欧洲中世纪背景的角色叫"李明" |

**关键规则**：确定小说的时代背景后，先研究该时代的命名习惯（用字偏好、单双名比例、姓氏分布、排辈规律），再为角色取名。名字本身就是世界观的一部分。

### 命名技巧

- **参考真实姓名库**：族谱、百家姓、历史人物名字的用字规律——用真实世界的名字训练自己的命名直觉
- **同一家族/势力的命名要有内在规律**：比如同辈用同一个字、同族用类似的音韵——这增加真实感，但规律本身不应暗示阵营（邪教家族的排辈字也可以是"文/承/永"）
- **不同阶层用不同的命名风格**：富贵人家和农户的取名逻辑不同——但"不同风格"不等于"名字泄露阶层"，农户也有文化人起的好名字
- **用反差制造惊喜**：最平凡的名字给最不凡的角色，最文气的名字给最粗犷的角色——这种反差本身就是好的叙事
